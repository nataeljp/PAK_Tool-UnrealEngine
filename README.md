COMO USAR
==================================================================

1. ABRIR PAK
   - Clique em "Abrir .PAK"
   - Selecione o arquivo

2. VISUALIZAR ARQUIVOS
   - Duplo clique para visualizar
   - Botao direito para menu de opcoes

3. EDITAR ARQUIVOS .INI
   - Duplo clique no arquivo .ini
   - Edite o conteudo
   - Salve com Ctrl+S ou botao Salvar

4. ADICIONAR NOVOS ARQUIVOS
   - Clique em "Adicionar"
   - Selecione os arquivos
   - Escolha o caminho interno no PAK

5. SUBSTITUIR ARQUIVOS
   - Selecione o arquivo no PAK
   - Clique em "Substituir" ou botao direito
   - Escolha o novo arquivo

6. DELETAR ARQUIVOS
   - Selecione o arquivo
   - Clique em "Deletar" ou botao direito
   - Confirme a delecao

7. SALVAR MODIFICACOES
   - Clique em "Salvar PAK Como"
   - Escolha o nome do novo arquivo
   - Todas as modificacoes serao aplicadas

8. CRIAR PAK NOVO
   - Clique em "Novo PAK"
   - Selecione a pasta com arquivos
   - Escolha o nome do PAK

FLUXO DE TRABALHO TIPICO
==================================================================

MODIFICAR ARQUIVO .INI:
1. Abra o PAK
2. Duplo clique no arquivo .ini
3. Edite o conteudo
4. Salve (Ctrl+S)
5. Feche o editor
6. Clique em "Salvar PAK Como"
7. Pronto! Novo PAK criado

ADICIONAR NOVOS ARQUIVOS:
1. Abra o PAK
2. Clique em "Adicionar"
3. Selecione os arquivos
4. Digite o caminho interno (ex: Config/)
5. Clique em "Salvar PAK Como"
6. Pronto!

SUBSTITUIR TEXTURA:
1. Abra o PAK
2. Encontre a textura
3. Botao direito -> Substituir
4. Selecione a nova textura
5. Clique em "Salvar PAK Como"
6. Pronto!
